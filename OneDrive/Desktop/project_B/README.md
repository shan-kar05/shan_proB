# Skill-Based Career Guidance (Local)

Simple Flask app that recommends career roles based on selected skills.

Quick start

1. Create a Python environment and install dependencies:

```powershell
python -m pip install -r requirements.txt
```

2. Run the app:

```powershell
python app.py
```

3. Open http://127.0.0.1:5000 in your browser.

Files

- [app.py](app.py) — Flask backend and recommendation logic
- [templates/index.html](templates/index.html) — Frontend form and results
- [static/style.css](static/style.css) — Basic styles
- [static/app.js](static/app.js) — JS to POST skills and show careers
